# Sri Krishna Modern Rice Mill — Website

A premium React + Three.js website for Sri Krishna Modern Rice Mill, Puducherry.

## Features

- 🌾 **Three.js Hero** — Golden rice grain particle system with paddy stalks
- 🌱 **Scroll-driven Paddy Growth** — A single paddy plant grows from seed → sprout → vegetative → booting → harvest as you scroll, with timeline cards on both sides
- 📊 **Animated Stats Counter** — Numbers count up when in view
- 🎨 **Rich color theme** — Warm gold, paddy green, teal, coral — inspired by your brand palette
- 📱 **Fully responsive** — Clean on desktop and mobile
- ⚡ **Smooth scroll animations** — Reveal on scroll throughout

## Color Theme
- Deep Gold `#D4A017` — heritage & quality
- Paddy Green `#4A7C3F` — fresh & natural
- Teal `#2A9D8F` — global & modern
- Coral `#E8623A` — warmth & energy

## Sections
1. **Navbar** — Fixed, glass-morphism on scroll
2. **Hero** — Three.js particle field + welcome copy
3. **About** — 4 value cards with hover animations
4. **Paddy History** — Scroll-driven 3D paddy growth + 5 timeline stages
5. **Stats** — Animated counters (65+ years, 45T/day, 150 staff, etc.)
6. **Products** — 6 featured varieties + VIP Krishna brand spotlight
7. **Quality** — 6-step process grid
8. **Customers** — 9 client logos + domestic/export split
9. **Contact** — Form with success state
10. **Footer** — Links, product list, tagline

## Setup

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Build for production
```bash
npm run build
```

## Tech Stack
- React 18
- Three.js (via npm)
- Framer Motion (available, not used yet — add for extra polish)
- Google Fonts: Playfair Display + Raleway
- Pure CSS animations (no heavy animation library needed)
